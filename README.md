# HTMLNotes

An interactive, premium dark-themed presentation deck containing comprehensive study notes and live simulator widgets covering the core components of HTML & HTML5 web development.

## Project Structure

*   **`index.html`** - Structural layout containing the 13 slide nodes and container wrappers.
*   **`styles.css`** - Premium dark mode presentation style with Glassmorphic design, custom responsive grids, and animations.
*   **`script.js`** - Navigation engine (keyboard controls, progress indicator, sidebar index) and custom widget simulator logic.

## Features

1.  **Evolution Timeline**: Interactive historical timeline selector for HTML versions (1991 - 2014).
2.  **DOM Tree Explorer**: Visual interactive explorer of HTML document structure, tags nesting, and attributes.
3.  **Link Simulator**: Real-time configuration and routing playground for anchor links (`_self`, `_blank`, `tel`, `mailto`, `download`).
4.  **Grid Planner**: Interactive `colspan` and `rowspan` visualizer for building table cell merges.
5.  **Interactive Code Playground**: Built-in sandbox code editor with live frame renderer and terminal logger.
